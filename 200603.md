테스트하기 좋은 코드란?
---

```javascript
function render(count) {
  const newCount = (value) => {
    newCount = count + 1; 
    render(newCount);  
  }
}

```
함수 자체로 테스트 가능해야 한다. 

예를 들어, 위와 같이 외부에 있는 변수 ```count```를 참조한다. 즉 함수가 함수 내부 파라미터에 의해 반환값이 결정되지 않고 외부에 의존하고 있다는 것이다. 이런 경우에는 테스트가 어려운다.
